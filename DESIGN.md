# Design

## System

Product UI, blue and white YouChat client style.

## Color

- Primary: `#168df2`
- Primary deep: `#0d63c7`
- Background: `#edf4ff`
- Surface: `#ffffff`
- Text: `#172033`
- Muted: `#68758a`
- Border: `#d9e2ef`

## Typography

Use the local Chinese system stack: `"Microsoft YaHei", "PingFang SC", "Segoe UI", Arial, sans-serif`.

## Components

- Login card follows the Windows client screenshot: compact labels, 33px controls, single blue connect button.
- Workbench uses three product regions: conversation list, chat surface, right-side tool tabs.
- Right toolbar tabs are `用户`, `快捷回复`, `订单`, `明细`, `聊天记录`.
- Radius stays near 5 to 6px to match the desktop client.

## Motion

Use short hover and active feedback only. No decorative page-load animation.
